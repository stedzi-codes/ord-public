# Oslo Roller Derby

Static Next.js site for Oslo Roller Derby.

## Getting Started

Install dependencies with pnpm:

```bash
pnpm install
```

Run the development server only when actively checking or debugging the site:

```bash
pnpm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
pnpm run lint
pnpm run build
pnpm run start
```

## Project Structure

Page routes live in `src/app`.

Page-level views live in `src/views`.

Reusable UI components live in `src/components`.

Static images live in `public/images`.
